== Description ==

This plugin provides several actions which affects the behaviour of a post entry which can be scheduled. The basic actions for this plugin are:

* Stick Post
* Unstick Post
* Private Post
* Draft Post
* Trash Post
* Delete Post
* Update Post Date
* Open Comments
* Close Comments
* Open Comments
* Close Comments
* Change Title
* Add Term
* Delete Term
* Add Meta
* Update Meta
* Delete Meta

This plugin is available in following languages

* English
* German

> As developer you might be interested in the API documentation which you can find at [GitHub](https://github.com/dasllama/Scheduled-Content-Actions/wiki/Developers-Documentation)

If you like this plugin you can [donate](https://paypal.me/HerrLlama) something and/or give this plugin a good review :)

== Installation ==

1. Just download, install and activate this plugin. There are no options.
